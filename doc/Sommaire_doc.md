## 2. Swagger

Nous mettons à disposition un **Swagger** pour la documentation de l'API

### Instructions pour lancer Swagger :
1. Download les dépendances :

```
npm install
```
2.  Lancer le serveur Express :
```
node app.js
```

Doc dispo sur le port 3000 !
